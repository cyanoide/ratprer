# Écran RATP - Temps Réel (ratprer)

Ce projet affiche les horaires en temps réel pour :
- **RER B** : Massy - Palaiseau & La Croix de Berny
- **Tram T10** : La Croix de Berny & Parc des Sports

## Fonctionnalités
- Design sombre type "Écran Gare".
- Affichage des codes missions (RER).
- Calcul du temps d'attente (prochain train, "+2 min", etc.).
- Changement de station via menu déroulant.

## Installation
Ouvrez simplement `index.html` dans votre navigateur.
